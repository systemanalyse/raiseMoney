# 软件设计文档

## 引言

### 编写目的

    本文档主要为了方便前后端开发者、数据库开发者协调对接，将各个系统模块拆解展示内部结构，使开发者对整个软件架构体系有一个清晰的认识，有利于系统开发和后面运营、维护工作的进行。

### 系统说明
