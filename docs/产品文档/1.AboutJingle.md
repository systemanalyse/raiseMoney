# 1. About Jingle

## 1.1 概述

| **产品名称**   | Jingle                                                       |
| :------------- | :----------------------------------------------------------- |
| **产品Slogan** | 找人帮忙，Jingle一下～                                       |
| **产品定位**   | 一款针对于在校大学生，基于校园实名认证的互助赚钱app          |
| **产品描述**   | 通过Jingle,同一学校的在校大学生可以在基于校园实名认证的信任基础上 |
| **关键词**     | 大学生 互助 挣闲钱 跑腿                                      |
| **目标人群**   | 在校大学生                                                   |
| **使用场景**   | 找人跑腿，找人线上帮忙，利用校园生活中的空余时间“顺便”挣闲钱 |
